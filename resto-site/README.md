# Maison Aurore — Site vitrine restaurant

Projet personnel réalisé en HTML / CSS / JavaScript vanilla.

## Objectif

Créer un site vitrine complet et responsive pour un restaurant gastronomique fictif, en appliquant les bonnes pratiques HTML/CSS vues en autodidacte.

## Pages / Sections

- **Accueil** — Hero pleine page avec call-to-action
- **Notre histoire** — Présentation du restaurant et stats
- **Menu** — Carte interactive avec onglets (Entrées, Plats, Desserts, Vins)
- **Galerie** — Grille CSS responsive avec effets hover
- **Contact & Réservation** — Formulaire complet avec validation JS

## Compétences mises en pratique

- Structure HTML5 sémantique
- CSS Variables, Flexbox, CSS Grid
- Design responsive (mobile, tablette, desktop)
- JavaScript vanilla : DOM, événements, IntersectionObserver
- Accessibilité de base (aria-label, labels de formulaire)
- Animations CSS (keyframes, transitions)
- Navigation sticky avec effet au scroll
